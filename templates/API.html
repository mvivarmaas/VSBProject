The following is the API for developers

Design is to be a simple and free look up for McGill Classes


ROOT URL

freeseatfinder.com

URL GET

/API/{ term }/<{Subject Code + - + Subject number} > /API/202109/COMP-202

Note:

Subject code / number should be the same as minerva

term is int. term = year + semester start month

Fall = 09, Winter = 01. Example Fall 2021 = 202109

Required: Subject code and Subject number


Success Response:

Code : 200

Content example :

{ "className" : "comp-202", "term" : 2019 "classTitle" : "Foundations of Programming" "classDescription" : "Introduction to computer programming in a high level language . . . ", "crnArray" : [2720, 2721, . . . ], "timeBlock" : [ { "crn" : 2720, "instructor" : "Campbell, Jonathan", "seatsAvailable" : True, "Section": 1, "Type": "Lecture", numberAvailableSeats: 5, numberAvailableWaitList: 2, numberMaxWaitList: 20 classTimes : Monday : 1:35 PM - 2:25 PM } {"crn" : 2721 , . . . } . . . ] }

Failed Response:

Code : 400

Bad Request

{ error : Bad request try entering correct subject code and title example "/API/COMP-206" }

Code : 404

Class not found

{ error : Input class not found, please try again with valid class }

Code : 429

Class not found

{ error : Rate limited too many requests }
